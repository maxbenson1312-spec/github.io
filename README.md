# Max Benson | Personal Portfolio Website

A multi-page personal portfolio website built from scratch using HTML, CSS, and JavaScript.

## Pages

- **index.html** — Main portfolio page with about, skills, projects, and contact sections
- **resume.html** — Full interactive resume with timeline layout and print functionality
- **scratch.html** — Scratch page on the world of hiking, featuring an embedded YouTube video and interactive Tableau sales dashboard
- **app.html** — Fitness Simulator web app with a BMI calculator, daily calorie calculator (Mifflin-St Jeor equation), and personalized workout plan generator

## Built With

- HTML5 & CSS3
- JavaScript (vanilla)
- Bootstrap 5
- Google Fonts
- Tableau Public (embedded dashboard)
